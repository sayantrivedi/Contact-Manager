Steps to Run the Application:

1. Run the scripts of "DBScript.sql" file under the "DATABASE SCRIPTS" folder in SQL SERVER.

2. Open the "ContactManager.sln" in Visual Studio.

3.Change the ConnectionString inside the Web.config fle of the "ContactsAPI" project and insert your MS SQL informations there.

4. In visual studio, set the "ContactManager" project as startup project and run the application.
